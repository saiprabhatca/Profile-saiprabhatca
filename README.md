# Sai Prabhat C A — Portfolio

This is a simple, single-page personal portfolio site built with **HTML** and **CSS** only (no frameworks). It showcases my skills, projects, education, and contact information.

## Files

- **`index.html`** – Main portfolio page (hero, about, skills, projects, education, contact, footer).
- **`project-section.html`** – HTML/CSS projects page linked from the Projects section.
- **`hardware-coming-soon.html`** – Coming soon page for hardware projects.

## How to run

1. Download or clone this project to your computer.
2. Open `index.html` directly in any modern browser (Chrome, Edge, Firefox, etc.).
3. Make sure you are connected to the internet so that Google Fonts load correctly.

## Editing content

- **Name, hero text**: Edit the hero section in `index.html`.
- **Skills**: Update the Skills section (`Languages`, `Hardware`) in `index.html`.
- **Projects**:
  - HTML/CSS projects live in the Projects section of `index.html` and the dedicated `project-section.html`.
  - Hardware projects currently point to `hardware-coming-soon.html`.
- **Contact links**: Email, LinkedIn, and GitHub are configured in the Contact section of `portfolio.html`.

## Notes

- The site is designed to be **responsive** and works on desktop and mobile.
- The **Download CV** button is intentionally disabled and shows a "Coming soon" tooltip on hover.
